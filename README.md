## VantaWire
A modular cybersecurity suite designed to automate network monitoring, intruder tracking, and system hardening. 

## 🛡️ Project Overview
VantaWire is a personal engineering project focused on developing automated tools for system security. As a student of AI Software Engineering, this project serves as a sandbox for implementing security concepts, hardening scripts, and network visibility tools.

## 🚀 Features
- **Network Visibility:** Automated scripts to monitor and log active network connections.
- **System Hardening:** Patch scripts designed to secure vulnerable system configurations.
- **Intruder Detection (In Development):** Logic for identifying unauthorized access attempts and generating logs for analysis.

## 🛠️ Tech Stack
- **Languages:** Bash, Python
- **Focus:** Automation, Cybersecurity, Linux System Administration

## 📋 Roadmap
- [ ] **Phase 1:** Standardize the network monitoring script to log real-time IP connections.
- [ ] **Phase 2:** Integrate automated alerts for unauthorized IP addresses.
- [ ] **Phase 3:** Develop an AI-driven log parser to flag suspicious activity patterns.

## 📝 Usage
Currently, the tools are designed for Linux-based environments.
1. Clone the repository: `git clone [your-repo-url]`
2. Navigate to the `/scripts` directory.
3. Run the module: `./monitor.sh`
